# Пример конспекта — выходная форма

**Вход:** Статья «Attention Is All You Need» (Vaswani et al., 2017)
**Тип:** experimental
**Дисциплина:** Компьютерные науки

---

## Введение

Авторы ставят перед собой задачу предложить новую архитектуру нейронных сетей для задач машинного перевода, полностью основанную на механизме внимания (attention), без использования рекуррентных или свёрточных слоёв. Актуальность обусловлена вычислительной неэффекностью RNN-подходов при параллелизации.

## Методы

Предложена архитектура **Transformer**, основанная исключительно на механизме multi-head self-attention. Ключевые компоненты:
- Multi-head attention (8 голов)
- Positional encoding (sinusoidal)
- Encoder-decoder структура (6 слоёв each)
- Датасеты: WMT 2014 EN-DE (4.5M пар), WMT 2014 EN-FR (36M пар)
- Метрика: BLEU

## Результаты

- Англо-немецкий: **28.4 BLEU** (prev SOTA: 26.4)
- Англо-французский: **41.0 BLEU** (prev SOTA: 39.9)
- Значительно меньшее время обучения по сравнению с RNN-аналогами

## Выводы

Авторы приходят к выводу, что architecture, основанная исключительно на attention mechanism, превосходит рекуррентные и свёрточные модели в задачах sequence-to-sequence, при этом значительно эффективнее в обучении. Ограничения: не рассматриваются задачи beyond NLP.
