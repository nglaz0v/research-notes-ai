# ИИ-ассистент для конспектирования научных статей

## 🎯 Цель проекта:
Создание нейроассистента, который поможет быстро и грамотно анализировать научные статьи, выделять ключевые тезисы, структурировать материал и формировать вопросы для самопроверки.

## 💼 Какие бизнес-задачи закрывает:
- Автоматизация обработки учебных и научных текстов
- Стандартизация формата конспектов для разных дисциплин
- Генерация проверочных вопросов по прочитанному материалу
- Экономия времени студентов и преподавателей на подготовку

## 🛠️ Функциональные требования:
1. Разработка системного промпта:
   – Роль, цель и стиль общения (академический, лаконичный, без воды)
   – Структура вывода (введение, методы, результаты, выводы, вопросы)
   – Ограничения (не придумывать данные, явно указывать на отсутствие информации в тексте)
   – Примеры промптов для разных типов статей (обзорные, экспериментальные, теоретические)
2. Подготовка базы знаний:
   – Шаблоны конспектов по дисциплинам
   – Готовые формулировки для выделения гипотез, выборок и метрик
   – Банк типовых вопросов для самопроверки
3. Документирование:
   – Системный промпт и сценарии — в Google Docs
   – База знаний с шаблонами — в Google Таблице или CSV
   – Инструкция по адаптации ассистента под новую дисциплину

## ⚙️Технические требования:
- Язык программирования: Python
- Платформа: OpenAI / GigaChat / YandexGPT / Ollama
- Формат сдачи: Google Docs
- База знаний: таблица в Google Таблице или CSV
- Пример структуры итогового конспекта и вопросов
- Инструкция по настройке под разные дисциплины
